# Computer Architecture Memory Project
Explore the memory hierarchy of the Raspberry Pi 5 and evaluate and impact software prefetching on program performance. The project profiles memory latency across cache levels, implement a simple software prefetching technique, and analyze the effects on performance metrics.

## Hardware
 * Raspberry Pi 5 - 8gb
 * Heatsink
 * Fan

## Software
 * Raspberry PI OS
 * C++
